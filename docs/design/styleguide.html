Rules of the system

#rule
1One accent. Merge purple carries focus, match echo, and the highlight bar — nothing else is purple.
2Monospace = identifier. Logins, repo paths, and the contract footer only; never body text.
3Kind is never encoded by color alone: shape (circle avatar vs { } tile) + mono path + KIND column.
4The dropdown footer narrates the contract (min chars, cap, sort, keys) — signature element, present in every open state.
5Motion: one 120ms dropdown entrance + loading pulse, both behind prefers-reduced-motion.
6Component styles only via --ac-* custom properties with baked-in fallbacks; demo canvas styling never leaks into the component.
