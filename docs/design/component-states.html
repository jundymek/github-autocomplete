Echoes the query, offers a next step. Announced via the live region.

07 error network / HTTP failure

Search failed
GitHub didn’t respond. Check your connection and try again.

All-or-nothing: if either of the two requests fails, the whole state is an error (owner decision U2) — a partial list would break the alphabetical contract. Retry re-fires the last query.

08 rate limit HTTP 403 with retry-after

GitHub rate limit reached
Unauthenticated search allows 10 requests per minute. Try again in 42s, or add a token to raise the limit.

Distinct from generic error: amber, names the cause, counts down from retry-after, points to the token option. The most likely state an evaluator hits — treat it as a first-class citizen.
